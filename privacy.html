Privacy Policy

Last updated: April 26, 2025

Short version: HarvestIQ collects nothing. Your API keys stay in your browser. Comments are analyzed in memory and immediately discarded. We have no servers that receive your data.

What HarvestIQ does

HarvestIQ is a Chrome browser extension that fetches public comments from YouTube videos and Reddit threads, sends them to an AI provider of your choice for analysis, and displays the results in the Chrome side panel.

All processing happens locally in your browser. HarvestIQ has no backend server of its own.

Data we do not collect

  • We do not collect, store, or transmit any personal information
  • We do not use analytics, telemetry, or crash reporting
  • We do not track which pages you visit or which analyses you run
  • We do not store your API keys on any server we control
  • We do not retain any comment data after analysis is complete

API keys

HarvestIQ requires API keys for AI providers (Anthropic, OpenAI, or Google Gemini) and optionally a YouTube Data API key. These keys are:

  • Stored exclusively in chrome.storage.local — your browser's local storage
  • Never transmitted to HarvestIQ or any server we operate
  • Sent only to the AI provider you selected (e.g. api.anthropic.com), exactly as any app using that provider would

Comment data

When you click Analyze, HarvestIQ fetches up to 100 public comments from the page you are viewing. These comments are:

  • Fetched from publicly accessible APIs (YouTube Data API v3, Reddit public JSON)
  • Held in memory only for the duration of the analysis request
  • Sent to your chosen AI provider along with your API key — governed by that provider's privacy policy
  • Discarded immediately after results are displayed — not written to disk or any database

Read-only access

HarvestIQ only reads public data. It never posts, votes, comments, or interacts with any platform on your behalf. One API call is made per user-initiated analysis — there is no background polling or scraping.

Usage count

To enforce the free-tier limit, HarvestIQ stores a monthly analysis count in chrome.storage.local. This counter exists only on your device and resets at the start of each month. It is never sent anywhere.

Payments

Paid upgrades are processed by ExtensionPay. HarvestIQ does not handle payment information directly. Please refer to ExtensionPay's privacy policy for details on how payment data is handled.

Third-party services

When you use HarvestIQ, your data may be sent to the following third parties — solely as a result of your own actions and API key configuration:

  • Anthropic — if you choose Claude as your AI provider
  • OpenAI — if you choose GPT as your AI provider
  • Google — if you choose Gemini as your AI provider, or use the YouTube Data API
  • Reddit — to fetch public comment threads you navigate to
  • ExtensionPay — if you purchase a Pro or Max license

HarvestIQ is not responsible for the privacy practices of these services.

Changes to this policy

If this policy changes materially, we will update the date at the top of this page. Continued use of the extension after changes constitutes acceptance of the updated policy.
